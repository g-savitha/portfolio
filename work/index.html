Authenticity Meter

A Password assistant engine which assists the user in setting up a robust password. This algorithm accepts user password, encodes(utf-8) it before encrypting it into SHA1. The encrypted password would be converted to hex format and this would be compared with historical persistent storage of `haveibeenpwned` API.
